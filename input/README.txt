===========================================
  INPUT FOLDER - Place Your Images Here
===========================================

This folder is where you should place images you want to process.

SUPPORTED FORMATS:
- JPG / JPEG
- PNG
- BMP
- TGA

HOW TO USE:
1. Copy your images into this folder
2. Run the image processor program
3. Select option 1 to browse and choose an image by number
4. Choose your processing operation
5. Processed images will be automatically saved to the 'output' folder

EXAMPLE:
- Place: photo.jpg in this folder
- Run program and select option 1
- You'll see: "1. photo.jpg"
- Enter: 1
- Image loads automatically!
