===========================================
  OUTPUT FOLDER - Processed Images Saved Here
===========================================

All processed images are automatically saved to this folder.

FILE NAMING:
Processed images are saved with descriptive names:
- Original: photo.jpg
- After grayscale: photo_grayscale.jpg
- After blur (size 5): photo_boxblur_5.jpg
- After edge detection: photo_sobel.jpg

ORGANIZATION:
- All output files are organized in this folder
- Original filenames are preserved with operation names appended
- Easy to find and compare results
